## Getting Started

- Clone the repository using ```git clone git@github.com:iam-Akshat/todolist-js.git```
- cd into the cloned repository with ```cd todolist-js```
- Run ```npm i```
- Run ```npm run watch```
- Open the index.html file in the 'dist' folder.
